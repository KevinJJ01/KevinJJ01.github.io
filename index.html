¡ Mis Habilidades y Conocimientos !

Planificación y Análisis de Requisitos

Se define el problema que el software debe resolver y se recogen los requisitos del usuario (realización del levantamiento de la información).

Diseño

Se crea la arquitectura del software, se especifican sus componentes y cómo interactuarán entre sí (se diseñan diferentes diagramas de diseño de proyecto).

Desarrollo o Implementación

Se desarrolla el código según el diseño previamente establecido (tanto front-end como back-end).

Pruebas (Testing)

Se verifican y validan las funcionalidades del software, detectando y corrigiendo errores.

Despliegue

El software se pone en producción y se entrega al usuario final.

Mantenimiento

Se realizan actualizaciones y mejoras, así como correcciones de errores a medida que el software se utiliza.

Manipulación de Bases de Datos SQL
